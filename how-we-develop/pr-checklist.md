## :eyes: Be aware of
- [ ] Infrastructure Change
- [ ] New Environment Variables
- [ ] DB Schema Change
- [ ] Seed Data
- [ ] New / Modification External Service Calls
- [ ] Incompatible Endpoint Interfaces (Params, Query, Body)
- [ ] Size of PR (> 400 LOC)
- [ ] New libraries
- [ ] More than 5 commits per PRs

------
## :kiss: Consider
- [ ] Feature Toggles
- [ ] Unit Tests
- [ ] Integration Tests
- [ ] Atomic Commit
- [ ] Split into smaller PRs
